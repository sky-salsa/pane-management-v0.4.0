    **Step 1: Implement path_decoder.rs:**

    ```rust
    use std::path::Path;
    use std::io::{BufRead, BufReader};
    use std::fs::File;

    /// Extract the `cwd` field from the first valid JSONL record in a project directory.
    /// This is the authoritative source for a project's actual filesystem path.
    /// DO NOT attempt to decode the encoded directory name algorithmically -- the encoding is lossy.
    pub fn extract_cwd_from_first_record(project_dir: &Path) -> Option<String> {
        // Find the first .jsonl file in the directory
        let entries = std::fs::read_dir(project_dir).ok()?;
        for entry in entries {
            let entry = entry.ok()?;
            let path = entry.path();
            if path.extension().map(|e| e == "jsonl").unwrap_or(false) {
                // Read first line only
                let file = File::open(&path).ok()?;
                let reader = BufReader::new(file);
                for line in reader.lines().take(3) {
                    if let Ok(line) = line {
                        if let Ok(val) = serde_json::from_str::<serde_json::Value>(&line) {
                            if let Some(cwd) = val.get("cwd").and_then(|v| v.as_str()) {
                                return Some(cwd.to_string());
                            }
                        }
                    }
                }
            }
        }
        None
    }
    ```

    Implement `parse_session_metadata(path: &Path) -> Result<SessionMeta, String>` that:
    1. Gets file size via metadata. If file is empty, return error.
    2. Reads the first 3 lines forward to get `sessionId` and `first_timestamp`.
    3. Uses `rev_lines::RevLines` to read backward from end of file:

          - Content does NOT start with `<command-` (command outputs)
          - Content does NOT start with `<local-command` (local command outputs)
       c. Extract `timestamp` from the last valid record found for `last_timestamp`.
    4. Truncate `last_user_message` to 200 characters.
    5. If any parse error occurs, return a partial result with what was successfully parsed. Only set `is_corrupted=true` if the file has zero parseable records.
