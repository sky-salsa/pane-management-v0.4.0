  <action>
    This task creates the project scaffold, installs all dependencies (npm + Cargo + Tauri plugins), and configures build tooling + permissions. No application code yet.

    1. Create the Tauri app using the official scaffolder:
       `npm create tauri-app@latest workspace-resume -- --template solid-ts`
       Run from the project root directory.

    2. Install frontend dependencies:
       ```
       cd workspace-resume
       npm install
       npm install -D tailwindcss @tailwindcss/vite
       ```

    3. Add Tauri plugins:
       ```
       npm run tauri add fs
       npm run tauri add store
       ```

    4. Edit `src-tauri/Cargo.toml` to add Rust dependencies under `[dependencies]`:
       ```toml
       notify = "6"
       rev_lines = "0.3"
       glob = "0.3"
       dirs = "5"
       ```
       Also ensure `tauri-plugin-fs` has the `watch` feature enabled:
       ```toml
       tauri-plugin-fs = { version = "2", features = ["watch"] }
       ```

    5. Configure Tauri FS permissions in `src-tauri/capabilities/default.json`:
       Add read scope for `$HOME/.claude/**/*` so the app can read session files.
       Add the following permissions to the capabilities array:
       - `"fs:allow-read-text-file"` with scope `["$HOME/.claude/**/*"]`
       - `"fs:allow-read-dir"` with scope `["$HOME/.claude/**/*"]`
       - `"store:allow-get"`, `"store:allow-set"`, `"store:allow-save"`

    6. Configure `vite.config.ts` to include the Tailwind plugin:
       ```ts
       import { defineConfig } from "vite";
       import solid from "vite-plugin-solid";
       import tailwindcss from "@tailwindcss/vite";

       export default defineConfig({
         plugins: [solid(), tailwindcss()],
         server: {
           port: 1420,
           strictPort: true,
         },
       });
       ```
  </action>
  <verify>
    <automated>cd workspace-resume && npm ls solid-js 2>&1 | tail -3 && cd src-tauri && cargo check 2>&1 | tail -5</automated>
  </verify>

    - File `workspace-resume/src-tauri/capabilities/default.json` contains `".claude"` in a scope
    - `cargo check` completes without errors in `src-tauri/`
